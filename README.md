# MouseSpot

プレゼンテーション・操作説明動画／画像の作成向けに、**マウスカーソル・クリック・キー入力を画面上で強調表示**する軽量な常駐ツールです。Windows 用・単一実行ファイル（約 350 KB）・ランタイム依存なし。

> Lightweight Windows overlay that highlights your mouse cursor, clicks, and keystrokes — built for presentations and screen-recording / tutorial captures. Single ~350 KB executable, no runtime dependencies.

---

## 特長

- **カーソルハイライト** … カーソル周囲に光（グロー＋リング）を表示
- **クリックエフェクト** … 左／右／中クリック・**ダブルクリック**（二重リング）・**ホイール**（方向インジケータ）・**ドラッグ軌跡**
- **キー入力表示** … 押したキー／ショートカット（例 `Ctrl + Shift + S`）を画面に表示
  - 「追加型（右へ並ぶ）」「上書き型」を選択可。表示位置はプリセット7種＋**座標指定**
- **スポットライト（減光）モード** … カーソル周囲以外を暗くして注目を誘導
- **設定画面** … 色・サイズ・透明度・表示時間などをGUIで調整、変更は即反映＆自動保存
- **軽量・非依存** … Rust + Win32 API 直叩きの単一 exe。レジストリ不使用
- **クリックスルー** … オーバーレイはマウス操作を一切妨げません

## 動作環境

- Windows 10 / 11（64bit）
- 管理者権限は不要

## 使い方

1. `mousespot.exe` を実行すると、タスクトレイに常駐します。
2. カーソルハイライトとクリック／キー表示が自動で有効になります。
3. **タスクトレイアイコンを右クリック**してメニューから操作できます。
   - ハイライト ON/OFF
   - スポットライト ON/OFF
   - 設定...
   - 終了

### ホットキー

| キー | 動作 |
| :--- | :--- |
| `Ctrl + Alt + H` | ハイライト表示のON/OFF |
| `Ctrl + Alt + S` | スポットライト（減光）のON/OFF |

### 設定

タスクトレイメニューの「設定...」から、**「マウス操作」「キー操作」** の2タブで各種項目を調整できます。

- 色（ハイライト・リング・各クリック・スクロール・ドラッグ・キー背景／文字）
- サイズ・不透明度・表示時間
- キー表示の方式（追加型／上書き型）・表示位置（プリセット／座標指定）
- スポットライトの半径・暗さ・縁ぼかし

設定は変更した時点で自動保存され、次回起動時に復元されます（保存先: `%APPDATA%\MouseSpot\config.toml`）。

## 録画ソフトでの使い方（重要）

オーバーレイはデスクトップ上に描画されるため、**OBS 等では「画面（ディスプレイ）キャプチャ」を使ってください**。

- ✅ **画面キャプチャ**（モニターを指定）→ ハイライト・クリック・キー表示がすべて録画に写ります。
  - 範囲を限定したい場合は、画面キャプチャに **クロップ（OBS のクロップ/パッドフィルタや Alt+ドラッグ）** を併用してください。
- ❌ **ウィンドウキャプチャ** → 指定ウィンドウのピクセルしか取り込まないため、別ウィンドウであるオーバーレイは**原理的に写りません**（これは OS の仕様で、同種のツールに共通の制約です）。

## ビルド

[Rust](https://www.rust-lang.org/)（stable）が必要です。

```sh
cargo build --release
```

`target/release/mousespot.exe` が生成されます。
